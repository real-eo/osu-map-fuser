osu! Map Fuser - SDL2 + OpenGL

Quick notes:
- The renderer currently uses OpenGL compatibility (legacy) calls for a minimal example.
- For production, migrate to modern OpenGL (GLAD/GL3) or Vulkan and implement per-view FBOs.

# Running & Compiling
## Running
Run `../run` in the MSYS2 MinGW shell from within `build/` folder to compile and run

## Compiling
Run `../compile` in the MSYS2 MinGW shell from within `build/` folder to compile and run
